## 二、问题详情与修复方案

### 2.1 登录页面配色问题

**当前代码**（[login_page.dart:89-122](file:///f:/GolangWorkspace/CryptographyDesign/frontendv2/lib/pages/login_page.dart#L89-L122)）：

```dart
// 背景：蓝色渐变
gradient: LinearGradient(
  colors: [Color(0xFF0D47A1), Color(0xFF1E88E5)],  // 深蓝色到蓝色
),

// 标题：黑色
Text(
  '企业财务审计系统',
  style: TextStyle(
    color: Colors.black87,  // 深灰色，与背景对比度低
  ),
),
```

**问题**：深蓝色背景 + 深灰色标题 = 视觉对比度不足

**修复方案**：将标题改为白色，提高对比度：

```dart
const Text(
  '企业财务审计系统',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    color: Colors.white,  // 改为白色
  ),
),
```

### 2.2 Recovery Code 复制按钮问题

**当前代码**（[audit_page.dart:150-156](file:///f:/GolangWorkspace/CCryptographyDesign/frontendv2/lib/pages/audit_page.dart#L150-L156)）：

```dart
IconButton(
  icon: const Icon(Icons.copy),
  onPressed: () {
    ScaffoldMessenger.of(context).showSnackBar(
      const SnackBar(content: Text('已复制到剪贴板')),
    );
  },
),
```

**问题**：按钮没有实际调用 `Clipboard.setData()`，只是显示提示

**修复方案**：使用 Flutter 的 `Clipboard` 类：

```dart
import 'package:flutter/services.dart';

IconButton(
  icon: const Icon(Icons.copy),
  onPressed: () async {
    await Clipboard.setData(ClipboardData(text: data.recoveryCode));
    if (mounted) {
      ScaffoldMessenger.of(context).showSnackBar(
        const SnackBar(content: Text('已复制到剪贴板')),
      );
    }
  },
),
```

### 2.3 Recovery Code 配色问题

**当前代码**（[audit_page.dart:133-145](file:///f:/GolangWorkspace/CryptographyDesign/frontendv2/lib/pages/audit_page.dart#L133-L145)）：

```dart
Container(
  decoration: BoxDecoration(
    color: Colors.grey[200],  // 浅灰背景
    borderRadius: BorderRadius.circular(8),
  ),
  child: SelectableText(
    data.recoveryCode,
    style: const TextStyle(
      fontFamily: 'monospace',
      fontWeight: FontWeight.bold,
      // 没有设置文字颜色，默认黑色
    ),
  ),
),
```

**问题**：灰色背景 + 黑色文字，对比度不够

**修复方案**：提高对比度，使用深色背景 + 浅色文字：

```dart
Container(
  decoration: BoxDecoration(
    color: Colors.grey[800],  // 深灰背景
    borderRadius: BorderRadius.circular(8),
  ),
  child: SelectableText(
    data.recoveryCode,
    style: const TextStyle(
      fontFamily: 'monospace',
      fontWeight: FontWeight.bold,
      color: Colors.white,  // 白色文字
      letterSpacing: 1.5,  // 增加字间距提高可读性
    ),
  ),
),
```

### 2.4 Recovery Code 掩码显示（建议功能）

**用户建议**：默认用掩码显示 Recovery Code，提供开关掩码按钮

**实现方案**：

```dart
void _showRecoveryCodeDialog(ItemSubmitResponse data) {
  bool isObscured = true;  // 默认掩码显示

  showDialog(
    context: context,
    barrierDismissible: false,
    builder: (context) => StatefulBuilder(  // 需要 StatefulBuilder
      builder: (context, setState) => AlertDialog(
        // ...
        Container(
          decoration: BoxDecoration(
            color: Colors.grey[800],
            borderRadius: BorderRadius.circular(8),
          ),
          child: Row(
            children: [
              Expanded(
                child: SelectableText(
                  isObscured
                      ? data.recoveryCode.replaceAll(RegExp(r'.'), '*')
                      : data.recoveryCode,
                  style: const TextStyle(
                    fontFamily: 'monospace',
                    fontWeight: FontWeight.bold,
                    color: Colors.white,
                    letterSpacing: 1.5,
                  ),
                ),
              ),
              IconButton(
                icon: Icon(isObscured ? Icons.visibility : Icons.visibility_off),
                onPressed: () {
                  setState(() => isObscured = !isObscured);
                },
              ),
              IconButton(
                icon: const Icon(Icons.copy),
                onPressed: () async {
                  await Clipboard.setData(ClipboardData(text: data.recoveryCode));
                  if (mounted) {
                    ScaffoldMessenger.of(context).showSnackBar(
                      const SnackBar(content: Text('已复制到剪贴板')),
                    );
                  }
                },
              ),
            ],
          ),
        ),
        // ...
      ),
    ),
  );
}
```

## 三、吐槽回应

> **[吐槽]** 不知道Agent是不是有意这样写的，生恐用户能看清

**回应**：这是无心之失。原始设计时只关注了功能实现和基本的配色协调（灰+黑是常见的安全码显示风格），但忽略了：
1. 在彩色背景（如蓝色渐变）上，深色文字的对比度问题
2. Recovery Code 作为安全敏感信息，应该使用高对比度配色

**改进措施**：
- 标题改为白色，与蓝色背景形成高对比度
- Recovery Code 容器改为深色背景 + 白色文字，模拟终端/代码风格
