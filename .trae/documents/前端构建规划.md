```dart
abstract class ApiResponse<T> {
  final int code;
  final String message;
  final T? data;

  const ApiResponse({required this.code, required this.message, this.data});

  factory ApiResponse.fromJson(Map<String, dynamic> json) {
    return ApiResponse(
      code: json['code'] as int,
      message: json['message'] as String,
      data: json['data'],
    );
  }

  bool get isSuccess => code == 0;
}
```

#### 3.4.3 Flutter桌面端Cookie支持说明

**重要说明**：Flutter桌面端（Windows）需要特殊配置以支持后端`Set-Cookie`响应头：

| 平台 | Cookie自动处理 | 说明 |
|------|--------------|------|
| Web | ✅ 自动 | 浏览器原生支持 |
| Android | ⚠️ 有限 | 需要配置CookieManager |
| iOS | ⚠️ 有限 | 需要配置CookieManager |
| Windows/macOS/Linux | ❌ 不支持 | 需要手动实现 |

**解决方案**：

1. **使用Dio的CookieManager拦截器**：
```dart
final dio = Dio();
dio.interceptors.add(CookieManager(CookieJar()));
```

2. **使用persist_cookie_jar持久化Cookie**：
```dart
// pubspec.yaml添加依赖
dependencies:
  dio: ^5.4.0
  cookie_jar: ^4.0.0
  persist_cookie_jar: ^4.0.0
```

3. **手动处理Set-Cookie响应头**：
```dart
// 登录请求后手动解析Set-Cookie
final response = await dio.post('/login', data: {...});
final cookies = response.headers['set-cookie'];
if (cookies != null) {
  for (var cookie in cookies) {
    await storageService.saveCookie('cookie', cookie);
  }
}
```

4. **后续请求携带Cookie**：
```dart
// 请求前添加Cookie到请求头
final cookie = await storageService.getCookie('cookie');
if (cookie != null) {
  dio.options.headers['Cookie'] = cookie;
}
```

**推荐方案**：使用`persist_cookie_jar`实现Cookie持久化，配合Dio拦截器自动处理Cookie的发送和接收。

### 3.5 步骤5：创建状态管理

**创建文件**: `lib/providers/auth_provider.dart`

```dart
final authProvider = StateNotifierProvider<AuthNotifier, AuthState>((ref) {
  return AuthNotifier();
});

class AuthState {
  final bool isAuthenticated;
  final User? user;
  final bool isLoading;
  final String? error;

  const AuthState({
    this.isAuthenticated = false,
    this.user,
    this.isLoading = false,
    this.error,
  });
}

class AuthNotifier extends StateNotifier<AuthState> {
  AuthNotifier() : super(const AuthState());

  Future<void> login(String username, String password) async {
    state = state.copyWith(isLoading: true);
    // 调用API
    state = state.copyWith(isLoading: false, isAuthenticated: true);
  }

  Future<void> register(String username, String password, String email) async {
    state = state.copyWith(isLoading: true);
    // 调用API
    state = state.copyWith(isLoading: false);
  }

  Future<void> logout() async {
    // 调用API
    state = const AuthState();
  }
}
```

## 7. 风险与注意事项

### 7.1 注意事项

1. **Cookie认证**: 后端使用HTTP-only Cookie，前端需要确保请求携带Cookie
2. **Token有效期**: Cookie有效期为5分钟，需要处理过期重登录
3. **敏感数据存储**: 份额数据需要加密存储，使用`flutter_secure_storage`
4. **跨域问题**: 开发环境需要配置CORS

### 7.2 潜在风险

| 风险 | 描述 | 缓解措施 |
|------|------|----------|
| API不可用 | 后端服务未启动 | 使用Mock数据兜底 |
| 网络超时 | 网络不稳定 | 添加超时重试机制 |
| 数据安全 | 敏感数据泄露 | 使用安全存储 |
