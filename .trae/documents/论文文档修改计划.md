架构图包含：
- 前端层：Flutter 桌面应用 → 6 个 feature 模块 → Hive/SSE
- API 层：RESTful API → 路由分组（/v1, /v1/protected, /v2）
- 业务层：Controller → Logic → Shamir/加密
- 数据层：DAO → PostgreSQL
- 基础设施：JWT 认证 / OTLP 链路追踪 / 文件监控
